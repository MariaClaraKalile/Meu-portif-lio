Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma lista de compras grande ou pequena. O objetivo é criar um acesso virtual mais visual onde seja possível anexar os produtos que precisam ser comprados, fornecendo uma descrição breve de cada um, incluindo a quantidade e a marca.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e os produtos em uma seção flexível (.livros). Cada produto é destacado em seu próprio contêiner (.livro), mostrando uma imagem do produto, quantidade e marca desejada. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a visualização dos produtos principalmente para adolecentes e homens que têm dificuldade de comprar o que a mulher solicitou.

O projeto consiste em uma calculadora online simples, desenvolvida com HTML e CSS que realiza a solução de contas das principais operações. O objetivo é que a calculadora seja online e de fácil acesso, sendo ideal para Iphone, Ipad e Desktop.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e os botões em uma seção flexível (.botoes). Cada botão é destacado em seu próprio contêiner (.botão1), mostrando uma imagem da calculadora. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

Minha Biblioteca: Uma Webpage Personalizada

O projeto consiste em uma aplicação simples desenvolvida com HTML e CSS, com o objetivo de criar uma página de login interativa e visualmente atrativa. A página foi projetada para oferecer uma experiência de usuário agradável, combinando um design moderno com funcionalidades dinâmicas.

A interação do usuário com a tela de login é fundamental para a experiência. Cada elemento da página (como campos de entrada, botões e mensagens) possui um comportamento definido por scripts que reagem a eventos específicos. Por exemplo, ao clicar no botão "Entrar", o sistema pode validar as credenciais inseridas, alterar o estilo dos campos de entrada ou exibir mensagens de erro ou sucesso. Esse comportamento é crucial para garantir uma navegação intuitiva e eficiente.

As ferramentas utilizadas, como HTML e CSS, são fundamentais para construir a estrutura visual e interativa dessa tela de login. A simplicidade do código e a clareza no design permitem que o projeto seja acessível e fácil de personalizar, além de garantir que a experiência do usuário seja fluida e agradável.
